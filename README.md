# ReProject Tree Generator

**ReProject Tree Generator** — это простое расширение для Visual Studio Code, которое позволяет мгновенно получить текстовое представление структуры вашего проекта.

Идеально подходит для создания документации или **передачи контекста проекта в нейросети** (ChatGPT, Claude, и др.), чтобы они понимали, как устроены ваши файлы.

## Возможности

*   **Копирование в буфер обмена:** Мгновенно копирует дерево файлов, чтобы вы могли сразу вставить его в чат с ИИ или документ.
*   **Сохранение в файл:** Создает файл `structure.txt` в выбранной папке с полным деревом проекта.
*   **Полный обзор:** В отличие от аналогов, это расширение **не скрывает** файлы. Оно отображает абсолютно всё: `node_modules`, `.git`, скрытые файлы и конфигурации. Вы получаете полную карту проекта.
*   **Удобство:** Работает через контекстное меню проводника (ПКМ).

## Как пользоваться

1. Откройте ваш проект в VS Code.
2. В проводнике слева (Explorer) нажмите **правую кнопку мыши** на папку, структуру которой хотите получить (или на корень проекта).
3. Выберите одну из опций:
    *   **Скопировать структуру** — дерево сразу попадет в ваш буфер обмена.
    *   **Создать файл структуры** — в папке появится файл `structure.txt`.

## Пример результата

Независимо от того, скопировали вы текст или сохранили в файл, результат будет выглядеть так:

```text
MyProject/
├── src/
│   ├── components/
│   │   ├── Header.tsx
│   │   └── Footer.tsx
│   ├── utils/
│   │   └── api.ts
│   └── App.tsx
├── .gitignore
├── package.json
└── README.md
```

## Установка (из VSIX)

Так как это приватное расширение:

**Вариант 1:**
1. Нажмите `F1` (или `Ctrl+Shift+P`) в VS Code.
2. Введите **Extensions: Install from VSIX...** и нажмите Enter.
3. Выберите файл `reproject-tree-generator-1.0.0.vsix`.
4. Готово!

**Вариант 2:**
1. Откройте вкладку Extensions (Расширения) в VS Code.
2. Нажмите на три точки в правом верхнем углу.
3. Выберите **Install from VSIX...**.
4. Выберите файл `reproject-tree-generator-1.0.0.vsix`.
