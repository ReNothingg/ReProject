# ReProject Tree Generator

**ReProject Tree Generator** это расширение для Visual Studio Code, которое генерирует текстовое дерево структуры проекта, копирует его в буфер обмена или сохраняет в файл.

Расширение удобно для документации, code review и передачи структуры проекта в LLM-инструменты вроде ChatGPT или Claude.

## Возможности

- Копирование структуры проекта в буфер обмена.
- Сохранение структуры в файл внутри выбранной папки.
- Сортировка: папки идут раньше файлов.
- Ignore-настройки с поддержкой glob-паттернов.
- Предупреждение, если часть каталогов не удалось прочитать.
- Автоматическое исключение выходного файла из дерева на корневом уровне, чтобы повторный запуск не включал `structure.txt` в результат.

## Как использовать

1. Откройте проект в VS Code.
2. В Explorer нажмите правой кнопкой по папке или файлу.
3. Выберите одно из действий:
   - **Скопировать структуру**
   - **Создать файл структуры**

Если команда вызвана без выбранного ресурса, расширение использует корень первого открытого workspace-folder.

## Настройки

Добавьте параметры в `settings.json`:

```json
{
  "reproject.ignorePatterns": [
    "node_modules",
    ".git",
    ".DS_Store",
    "dist",
    "out",
    ".vscode",
    "__pycache__",
    "*.log",
    "coverage/**"
  ],
  "reproject.outputFileName": "structure.txt"
}
```

- `reproject.ignorePatterns` исключает файлы и папки из дерева.
- Поддерживаются простые glob-паттерны:
  - `node_modules` исключает каталог с таким именем на любом уровне.
  - `*.log` исключает файлы по имени.
  - `coverage/**` исключает корневой каталог `coverage` и всё его содержимое.
  - `.next/cache` исключает конкретный относительный путь.
- `reproject.outputFileName` задаёт имя файла для сохранения результата.

## Поведение при ошибках

Если корневую папку прочитать нельзя, команда завершится ошибкой.

Если недоступна только часть вложенных каталогов, расширение всё равно построит дерево по доступным данным и покажет предупреждение со списком проблемных путей.

## Пример результата

```text
MyProject/
├── src/
│   ├── components/
│   │   ├── Header.tsx
│   │   └── Footer.tsx
│   ├── utils/
│   │   └── api.ts
│   └── App.tsx
├── package.json
└── README.md
```
